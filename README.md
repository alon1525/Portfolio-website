# Alon Krumer — Portfolio

Personal portfolio site: React, Vite, and client-side routing. Project case studies, contact details, and CV download.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output is written to `dist/` (used by Vercel via `vercel.json`).

## Lint

```bash
npm run lint
```
